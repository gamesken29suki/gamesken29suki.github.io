# 🌸 gamesken29suki の Webサイト | My INFP-T World 
> 「心の声に耳を澄まし、優しく、温かいものを表現する場所。」
# プロジェクト概要
* これは、INFP-T（仲介者）である私が、日々の内省、創造的な活動、そして大切にしている価値観を共有するために構築したパーソナルWebサイトです。
* 訪問者がリラックスでき、内面と向き合えるような、暖色系で落ち着いたデザインを特徴としています。
* ブログ機能や趣味のギャラリーを通じて、私の多面的な世界観に触れていただけます。
# 主な特徴
* 🌙 ダークモード対応: 訪問者の設定や手動操作で、落ち着いた色彩（紫と暗色）に切り替え可能です。
* 🔗 スムーズなナビゲーション: ページ内リンクで、目的のセクションへ滑らかにスクロールします。
* 🗂️ ブログカテゴリフィルタ: 記事を「内省」「デザイン」「コミュニティ」などで簡単に絞り込めます。
* 🖼️ レスポンシブなギャラリー: 趣味や作品を美しく表示し、モバイルにも対応しています。
* ⚙️ 技術スタック: HTML5, CSS3 (カスタムプロパティ/Flexbox/Grid), Vanilla JavaScript
